Estructura de programación cíclica

Bloque Descripción
Start En Rogic, el comando start se utiliza para iniciar la ejecución del programa o simulación que has creado con bloques. Es básicamente el punto de partida desde donde se comienzan a ejecutar las instrucciones que has programado.
DC Motor Un DC Motor es un motor que usa electricidad para girar y hacer que algo se mueva. En los robots, se usa para mover las ruedas o alguna parte del robot. En Rogic, el bloque "DC Motor" sirve para decidir hacia dónde gira el motor (adelante o atrás) y a qué velocidad. Su función es ayudar a que el robot camine, gire o mueva algo.
Loop Loop significa bucle o repetición. En Rogic, el bloque "loop" se usa para que el robot repita una o varias acciones muchas veces. Sirve para que el programa no tenga que escribir lo mismo varias veces, y el robot haga una acción una y otra vez, o durante un tiempo.
On On es un bloque que sirve para encender algo del robot, como una luz o un motor. Se usa para activar un componente y que empiece a funcionar.
Delay Delay es un bloque que hace que el robot espere un tiempo antes de hacer otra acción. Sirve para pausar el programa durante unos segundos.
Off Off es un bloque que apaga lo que se encendió con el bloque "On", como una luz o un motor. Se usa para detener o apagar un componente del robot.
